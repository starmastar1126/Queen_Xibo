Installation

The Xibo Client should be installed on each display. It is the Xibo client that collects and displays the scheduled content on the output device (tv, projector, etc).

The Xibo Server should run on any platform that Apache, PHP and MySQL are available for. It has been tested on GNU/Linux (Ubuntu Precise 12.04) and Microsoft Windows platforms.

Prerequisites

Xibo Client Requirements

Before attempting to install the Xibo Client please ensure the following minimum prerequisites are met.

  • A network connection to the Xibo Server (possibly over the Internet)
  • Microsoft Windows XP/Vista/7
  • .NET Framework v3.5 and latest service packs
  • Internet Explorer 7 or later
  • Flash Player Version 11 32bit or later (for Flash Support)
  • Windows Media Player 11 or later

Minimum Recommended Hardware Specifications are:

  • Processor: 1.5GHz
  • RAM: 2GB
  • Hard drive Space: 40 GB

Xibo Server Requirements

The Xibo server requires a PHP enabled webserver (e.g. Apache), PHP v5.1.4 or later and MySQL, as well as some SQL extensions.

Minimum specification depends largely on the number of clients to be supported, and the frequency of client updates. Any hardware capable of running a reasonably modern Linux distribution, or Microsoft Windows 7/XP should be capable of supporting a few clients.

Xibo stores all the content you schedule for display on the server. Disk space is therefore a function of the amount of content you upload.
